Secure Saved Search API¶

The secure saved search API was developed in an attempt to utilize information about authenticated users in regards to saving searches on the microsites. It uses both CORS and JSONP in the similar fashion to Secure Blocks.

The Secure Saved Search API is currently used by the Saved Search Widget.

mysearches.views.secure_saved_search(*args, **kwargs)[source]¶

Handling for creation of a saved search via the saved search secure block widget.

Parameters:request – inbound request
Returns:Dictionary, to be converted by autoserialize decorator
